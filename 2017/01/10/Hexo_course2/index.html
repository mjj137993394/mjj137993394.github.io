发表于 2017-01-10 | 分类于 Hexo | | 本文总阅读量 次

上一篇介绍了 GitHub Pages 服务开启，这篇文章将介绍 Hexo 的安装和使用以及将博客部署到 GitHub Pages 的操作。
Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

一、Node.js 安装

Hexo 是基于 Node.js 安装，所以安装Hexo之前首先要安装 Node.js。下载地址：https://nodejs.org/en/download/ 根据自己的操作系统，下载对应版本即可。安装过程比较简单，一直下一步即可安装完成。
